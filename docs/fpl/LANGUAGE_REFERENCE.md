# FPL Language Reference

This file is a short index.

The canonical, implementation-accurate authoring reference is:

- `docs/fpl/POLICY_AUTHORING_REFERENCE.md`

Use that document for:
- grammar and supported constructs
- runtime expression context and helper functions
- identity/IdP and credential sequestration authoring patterns

- parsed-vs-enforced coverage boundaries
- CI validation and contributor checklists

Why this split exists:
- Earlier versions of this file mixed aspirational and implemented behavior.
- The canonical reference is now maintained as the normative source to avoid drift.
